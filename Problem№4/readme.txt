DNS-сервер
При запуске сервер сразу может появиться сразу только 3 сообщения:
- "Файл не был найден"
- "Произошла ошибка ..."
- "Кэш был загружен"
Если рядом со скриптом отсутствует файл "dns_cache.pkl", то выведет первое сообщение
Если "dns_cache.pkl" как либо искажен, то выведет второе сообщение, где вместо троеточия выведется точная ошибка
Иначе, выведется 3 сообщение.

При запуске сокет слушает 53 UDP-порт по адресу 0.0.0.0
При отправлении на него запроса он сохраняет запрос в кэш и отправляет запросы в корневые сервера A-D, после чего возвращает ответ.
